# **App Name**: Libre Chat

## Core Features:

- AI Chat Interface: Engage in free-form conversation with an integrated LLM tool, allowing users to send prompts and receive intelligent responses.
- Conversational History Display: A scrollable, clean display that chronologically shows the user's prompts and the AI's generated replies within the current session.
- User Prompt Input: A dedicated and intuitive text input field for users to type and submit their messages to the AI.
- Real-time AI Response: Display AI responses as they are being generated, enhancing user engagement and providing a dynamic conversational flow.
- Session Management: Allows users to easily clear the current chat history to start a fresh conversation with the AI.

## Style Guidelines:

- Primary color: A clear, refreshing sky blue (#4FC3F7) for primary interactive elements and branding, reflecting a modern and accessible feel.
- Background color: A very light, neutral grey-blue (#F7F8FA) providing an airy and spacious canvas, typical of clean web interfaces built with React and Tailwind CSS.
- Accent color: A warm, inviting coral orange (#FF8A65) for calls-to-action and important highlights, designed for clear visual hierarchy.
- Headline and body font: 'Inter', a highly readable, modern grotesque-style sans-serif, offering excellent legibility across various screen sizes, easily configurable with Tailwind's utility classes in a React application.
- Use a set of simple, crisp line-based icons for chat-related actions (send, clear, new conversation), ensuring a cohesive and minimalist aesthetic often achieved with icon libraries compatible with React/Tailwind.
- A clean, responsive chat interface layout, utilizing a utility-first approach (like Tailwind CSS) to ensure adaptability across devices, with a fixed input area at the bottom and a scrollable message history, implemented efficiently with React components.
- Subtle and smooth transition animations for message entries, loading states, and interactive elements, enhancing user experience without being distracting, implemented efficiently with CSS transitions/animations within React components.
